# 🚗 Car Animation Project

A simple car animation built using **HTML, CSS, and JavaScript**.
This project demonstrates basic animation techniques and movement effects.

---

## 📌 Features

* 🚘 Moving car animation
* 🌄 Background scrolling effect
* 🎬 Smooth animation using CSS / JavaScript
* 📱 Simple and clean design

---

## 🛠️ Technologies Used

* HTML
* CSS (Animations & Keyframes)
* JavaScript
